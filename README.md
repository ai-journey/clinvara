Clinvara MVP
AI-Assisted Eligibility Screening for Clinical Research Associates
Clinvara is a prototype Clinical Trial Eligibility Assistant that uses NLP and rule-based AI to:
Extract inclusion/exclusion criteria from protocol PDFs
Load and analyze synthetic patient datasets
Run automated eligibility matching
Provide explainability and metric dashboards
Support CRA overrides & notes
Export site-ready match reports
This MVP is part of a Capstone project and is designed for future extensibility.

Features (MVP):
Multi-study workspace system
Protocol upload + text parsing
AI-based criteria extraction
Patient dataset ingestion (flattened Synthea CSV)
Eligibility matching engine
Explainability per patient
CRA override workflow
Exportable results (CSV/JSON)
Model metrics panel
Bias/fairness checks
Full audit traceability
